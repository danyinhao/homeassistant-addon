# Markdown 示例文档

## 基本介绍

这是一个展示 Markdown 常用语法的中文示例文档。Markdown 是一种轻量级标记语言，适合编写文档、笔记等。

## 文本格式

- **加粗文本** （使用 `**文本**` 或 `__文本__`）
- *斜体文本* （使用 `*文本*` 或 `_文本_`）
- ~~删除线文本~~ （使用 `~~文本~~`）
- `行内代码` （使用 `` `代码` ``）
